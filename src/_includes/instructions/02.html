Prototyping

Let's prototype a page using USWDS. We're going to be using:

  • USWDS components
  • Theme settings
  • Tokens
  • Grid layout
  • Utility classes

Exercise

  1. Add the flag variant of USA Card to the news section.
  2. Add default Card to the news section.
  3. Add a headings only collection for latest blog posts section.
  4. Add events only collection for latest blog posts section.
  5. Add a two column layout starting at tablet-lg breakpoint. News should be in the main content and the rest in the sidebar.
  6. Remove instructions once finished.
